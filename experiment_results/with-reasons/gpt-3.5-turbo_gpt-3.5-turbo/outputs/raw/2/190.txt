New Prompt: Classify the following "Message" from chat in an online community into one of the specified categories, "toxic" or "clean". Provide a reason for the classification in the specified JSON format. Include the prediction result, which is not present in the original prompt. Utilize this information to design a new prompt without changing the output format.

Message: I'm tired of your nonsense.

### Examples:

#### Example 1:
Message: "You are so dumb!"

Output format:
```json
{
"message": "You are so dumb!",
"reason": "The message contains offensive language and is derogatory.",
"predicted": "toxic"
}
```

#### Example 2:
Message: "Thanks for your help!"

Output format:
```json
{
"message": "Thanks for your help!",
"reason": "The message expresses gratitude and positivity.",
"predicted": "clean"
}
```
